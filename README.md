# mlops-iris
Projet MLOps complet basé sur le dataset Iris, intégrant FastAPI, Streamlit, MLflow et Docker. Il permet d’entraîner plusieurs modèles, de suivre leurs métriques dans MLflow et de charger dynamiquement un modèle depuis l’interface utilisateur. Une architecture simple et reproductible illustrant les bonnes pratiques de déploiement et de suivi.
